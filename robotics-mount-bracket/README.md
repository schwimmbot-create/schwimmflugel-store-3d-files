# Robotics Mount Bracket — STL Pack

3D-printable mount bracket designed for hobby servos and small stepper
motors. Print in PETG or ABS for bench use; PLA is fine for fixtures
that won't see sustained heat or load.

This is the **v1.0.0** release of the Robotics Mount Bracket STL pack.
Released under [CC-BY-4.0](../../LICENSE).

## What's in this folder

| File | Format | Use it for |
|---|---|---|
| `robotics-mount-bracket.stl` | STL (ASCII) | Slicing directly |
| `robotics-mount-bracket.step` | STEP AP214 | Re-editing in Fusion 360 / SolidWorks / FreeCAD |

## Print settings (recommendation)

- **Material:** PETG or ABS (PLA acceptable for non-load fixtures)
- **Layer height:** 0.2 mm
- **Infill:** 30% gyroid or cubic
- **Perimeters:** 4
- **Orientation:** flat side down, mounting boss facing up

See [`print-profile.md`](./print-profile.md) for the full slicer notes
and the alternate M3 bolt-pattern instructions.
